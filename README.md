# 📖 Overview

Managing customer credit (Udhaar) manually is one of the biggest challenges for small businesses. Traditional notebooks often lead to lost records, delayed payments, poor cash flow, and inefficient follow-ups.

This platform provides a complete digital solution by combining **workflow automation, AI, and analytics** to simplify credit management and improve business operations.

## 🛠️ Technology Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend & Automation:** n8n Workflow Automation
- **Database:** PostgreSQL
- **AI:** Google Gemini API
- **Machine Learning:** Python, Scikit-learn *(Future Enhancement)*
- **Communication:** WhatsApp API / SMS API
- **Deployment:** Vercel, Render
- **Tools:** Docker, Git, GitHub
